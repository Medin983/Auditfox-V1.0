Wir möchten Sie darüber Informieren, dass auf dieser Website keine personenbezogenen Daten von Ihnen erhoben, verarbeitet oder gespeichert werden. Ihre Privatsphäre ist und ein wichtiges Anliegen.


Analyse von eingegebnene Urls

Die Kernfunktion unserer Website ist die Analyse von Urls, die sie selbst in ein Dafür vorgesehenes Eingabefeld eingeben. Diese Analyse erfolgt ausschließlich bezogen auf die Inhalte der externen, von Ihnen eingegebenen Website.


Wichtiger Hinweis

Wenn Sie eine URL eingeben, um diese analysieren zu lassen, wird diese URL temporär verarbeitet, um die Analyse durchzuführen. Diese eingegebene URL wird jedoch nicht dauerhaft gespeichert, mit anderen Daten verknüpft oder dazu verwendet, Rückschlüsse auf Ihre Person zu ziehen Die Analyseergebnisse beziehen sich ausschließlch auf die technische und inhaltliche Beschaffenheit der von Ihnen angegebenen Website.
